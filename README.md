## Что делает калькулятор

Вы вводите пары (T, U<sub>P</sub>) при охлаждении образца от 170 °C до комнатной температуры. Калькулятор автоматически:

- Вычисляет **σ, ln σ, 1000/T** и погрешности для каждой строки
- Строит **график ln σ = f(1000/T)** с выделением линейного участка и линией МНК
- Рассчитывает угловой коэффициент **b** методом парных точек (по разд. 8 методички)
- Находит **ширину запрещённой зоны E<sub>g</sub>** в джоулях и электронвольтах
- Считает **полную цепочку погрешностей** δσ → δb → ΔE<sub>g</sub>
- Сравнивает результат с табличным значением E<sub>g</sub> = 0.66 эВ
- Генерирует готовый **протокол** для отчёта

## Физика

### Удельная проводимость

```
σ = I · l / (A · U_P)
```

`I` — ток через образец [А], `l` — длина [м], `A = b·d` — площадь поперечного сечения [м²], `U_P` — продольное напряжение [В].

### Температурная зависимость

В области **собственной проводимости** (высокие температуры):

```
σ = σ₀ · exp(−E_g / 2kT)
```

После логарифмирования:

```
ln σ = ln σ₀ − (E_g / 2k) · (1/T)
```

На графике `ln σ = f(1000/T)` это прямая с угловым коэффициентом `b = −E_g / (2k)`.

### Рабочая формула

При построении графика по оси `1000/T` (вместо `1/T`):

```
E_g = −2k · (b / 1000)   [Дж]
E_g [эВ] = E_g [Дж] / 1.602×10⁻¹⁹
```

### Три области на кривой

| Область | Температура | Характер зависимости |
|---------|-------------|----------------------|
| I. Примесная | низкая T | нелинейный изгиб |
| II. Истощение примесей | умеренная T | переходная |
| III. Собственная | **высокая T** | **линейный участок ← используется** |

### Метод парных точек (раздел 8 методички)

```
b_i = (ln σ_{i+k} − ln σ_i) / (1000/T_{i+k} − 1000/T_i),   i = 1…k

σ⟨b⟩ = √( Σ(b_i − ⟨b⟩)² / k(k−1) )

Δb = t(P=0.90, f=k−1) · σ⟨b⟩
```

Коэффициент Стьюдента — при P = 0.90.

### Погрешности

```
δσ = √(δU_P² + δI² + δl² + δA²)   — относительная погрешность σ

δE_g = δb = Δb / |⟨b⟩|

ΔE_g = |E_g| · δb
```

## Порядок проведения эксперимента (по методичке)

1. Установить ток через образец **I = 5 мА** (ручка 1, индикатор 2, кнопка 7 нажата)
2. Выбрать режим температуры на модуле Холла (кнопка 7), включить нагрев (кнопка 12)
3. Дождаться нагрева до 170 °C, затем снимать показания **при охлаждении**:
   - 170 → 100 °C: через **10 °C**
   - 100 → 25 °C: через **5 °C**
4. При каждой температуре записывать U<sub>P</sub> в мВ
5. Своевременно переключать предел вольтметра (U<sub>P</sub> растёт экспоненциально)

## Как пользоваться калькулятором

### 1. Параметры
Проверьте геометрические размеры образца, ток и класс точности вольтметра.

### 2. Измерения
Введите пары (T, U<sub>P</sub>) в таблицу. Для каждой строки выберите предел вольтметра (200 мВ / 2 В / 20 В). Нажмите **Enter** для пересчёта строки. Жёлтый цвет ячейки U<sub>P</sub> — сигнал переключить предел.

### 3. Обработка
- Перейдите на вкладку **«Обработка»**
- С помощью слайдеров **выделите линейный участок** (высокие T, малые 1000/T)
- Смотрите таблицу МПТ и результат b ± Δb
- E<sub>g</sub> вычисляется автоматически

### 4. Протокол
Нажмите «Скопировать» — текст готов для вставки в отчёт.

## Формулы погрешностей

### МНК (контрольный расчёт)

```
b = (n·Σxᵢyᵢ − Σxᵢ·Σyᵢ) / (n·Σxᵢ² − (Σxᵢ)²)
```

### Метод парных точек

```
aᵢ = (y_{i+k} − yᵢ) / (x_{i+k} − xᵢ),   i = 1…k

σ⟨a⟩ = √( Σ(aᵢ − ⟨a⟩)² / (k·(k−1)) )

Δa = t(P=0.90, f=k−1) · σ⟨a⟩
```

### Погрешность σ

```
δσ = √(δU_P² + δI² + δl² + δA²)

δU_P = ΔU / U_P,   ΔU = (класс % / 100) × предел_В
δA ≈ √(δb² + δd²)
```
