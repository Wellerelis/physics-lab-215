Теоретические основы
Температурная зависимость электропроводности полупроводника
Удельная проводимость через ток и напряжение
σ = I · l / (A · UP)
I — ток [А], l — длина [м], A = b·d — сечение [м²], UP — продольное напряжение [В], σ [Ом⁻¹·м⁻¹]
Три области кривой ln σ = f(1/T)
I. Примесная проводимость — низкие T (изогнутый участок)
II. Истощение примесей — переходная зона
III. Собственная проводимость — высокие T → линейный участок
Экспоненциальный закон (область III)
σ = σ₀ · exp(−Eg / 2kT)
Eg — ширина запрещённой зоны [Дж], k = 1.381×10⁻²³ Дж/К
Логарифмирование → линейная зависимость
ln σ = ln σ₀ − (Eg/2k) · (1/T)
Наклон прямой: b = −Eg/(2k)
Рабочая формула (ось 1000/T)
Eg = −2k · b · 1000 [Дж]
b — наклон графика ln σ = f(1000/T); b·1000 переводит его из единиц шкалы 1000/T в Кельвины
Перевод в электронвольты
Eg [эВ] = Eg [Дж] / 1.602×10⁻¹⁹
Погрешности
δσ = √(δUP² + δI² + δl² + δA²) — относительная
δEg = δb = Δb / |⟨b⟩|
ΔEg [Дж] = |Eg| · δb,  ΔEg [эВ] = ΔEg [Дж] / e
δUP = ΔU / UP — меняется от строки к строке
ℹТабличное значение ширины запрещённой зоны нелегированного Ge: Eg = 0.66 эВ при T ≈ 300 К.
